Chat messenger plugin with a toolbar, with some useful extras.  Also have Facebook like and google plus one icons.

== Description ==
A chat messenger plugin, which gives Wordpress bloggers the option to add extra functionality to their blog, when installed gives a tool bar with an Msn messenger style application, also has Facebook like buttons and Google plus 1 buttons, so people can use this to like and +1 whatever page they are browsing at the time.  The messenger is free and you DONT need to register to use it. Its easy and fun to use and gives people the chance to talk to people from all over the world via this little application.

== Installation ==
* Extract worldofchat folder
* Upload the woc.php to the \'/wp-content/plugins/\' directory.
* Activate the plugin through the \'Plugins\' menu in WordPress.
* After Activation Goto dashboard/settings and then WOC Settings setting 
  and activate the plugin, check the first box for messenger and 2nd for help with sponsored link.

== Frequently Asked Questions ==
How does it work ?
It just installs some code into your blog, that makes the toolbar appear on all your site pages, it does not load straight away takes 5 seconds, so it will not allow your users, once loaded you can use the messenger and other features on the toolbar.

Is it FREE ?

Yes its free

Do you have to register to use the Messenger ?
No - guests can sign in, there is also an option to register for free

How does the Facebook like and Google plus 1 buttons work ?
Which ever pages users are browsing on your blog the toolbar will load, if a user click Face book like or google plus one, this page will be added to the appropriate Facebook and Google profile, users will be asked to sign in, if they are not.
